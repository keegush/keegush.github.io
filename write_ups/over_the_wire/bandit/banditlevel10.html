Over the Wire - Bandit Level 10

Level Information

  • Host Name: bandit.labs.overthewire.org

Level Goal

The password for the next level is stored in the file data.txt, which contains base64 encoded data.

Commands you may need to solve this level

  • grep,
  • sort,
  • uniq,
  • strings,
  • base64,
  • tr,
  • tar,
  • gzip,
  • bzip2,
  • xxd
* Not all commands listed will be needed
Command Explanation
grep Print lines matching a pattern
sort Sort lines of text files
uniq Report or omit repeated lines
strings Print the strings of printable characters in files
base64 base64 encode/decode data and print to standard output
tr translate or delete characters
tar The GNU version of the tar archiving utility
gzip compress or expand files
bzip2 a block-sorting file compressor, v1.0.6
xxd make a hexdump or do the reverse

https://explainshell.com is a good site to search up Ubuntu's man page repository.

Write up

  1. Step one would be to locate the data.txt file. Run the ls command to pull the list of files and folders. bandit9@bandit:~$ ls
    data.txt     Here is the file in the home directory.
  2. Let's take a peak inside the file and see what we are working with. bandit10@bandit:~$ tail data.txt
    VGhlIHBhc3N3b3JkIGlzIGR0UjE3M2ZaS2IwUlJzREZTR3NnMlJXbnBOVmozcVJyCg==
     So this is what base64 encoded data looks like. Let us look into how we can decode this file.
  3. Looking through our list of commands the choice seems pretty obvious. I will go to explainshell.com to figure out how to use the base64 command, be right back... Well there is a decode flag for this command so let's use the enchantation with the decode flag. bandit10@bandit:~$ base64 --decode data.txt
    The password is aaaaaaaaaaaaaaaaaaaaa
    And here is the flag that we wanted to capture to gain access into bandit level 11.
  4. And don't forget to use the login username of bandit11 for the next level.

* Please note that the password is not shown as per Over The Wire's request.

Learned:
