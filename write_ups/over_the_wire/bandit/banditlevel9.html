Level Goal

Alternate Solutions

  • After a bit more research I found a more advanced technique using some grep flags and regex. bandit9@bandit:~$ strings data.txt | grep -oP '={2,}\K.*'
    ========== the
    ========== password
    ========== is
    ========== aaaaaaaaaaaaaaaaaaaaa
  • And you can see that they give a nice little message pointing right to the password. How this grep -oP '={2,}.*' command works:
    • -o — print only the matched part
    • -P — enable Perl-compatible regex
    • ={2,} — match 2 or more = signs
    • .* — match everything to the end of the line
    And to clean it up even more and remove the '=' use this command grep -oP '={2,}\K.*' The \K — "forget" the = signs (reset the match start to here).

* Please note that the password is not shown as per Over The Wire's request.
