Level Goal

The password for the next level is stored in the file data.txt, where all lowercase (a-z) and uppercase (A-Z) letters have been rotated by 13 positions

Write up

  1. Step one, like usual, would be to locate the data.txt file. Run the ls command to pull the list of files and folders. bandit11@bandit:~$ ls
    data.txt     Here is the file in the home directory.
  2. Let's take a peak inside the file and see what we are working with. bandit11@bandit:~$ tail data.txt
    Gur cnffjbeq vf 7k16JArUVv5LxVuJfsSVdbbtaHGlw9D4
     Remember from the level goal that each character has been rotated 13 positions. We could break out an alphabet chart and start counting each back 13 spots but I don't think that is the purpose of this exercise.
  3. Looking through our list of commands it appears that the tr command is the one to go with but I will need to do some more research... I went to explainshell.com and looked at the man page for tr and that still didn't help clear things up. So, I did find a stack overflow answer that helped solve this problem. bandit11@bandit:~$ cat data.txt | tr 'a-z' 'n-za-m' | tr 'A-Z' 'N-ZA-M'
    The password is aaaaaaaaaaaaaaaaaaaaa
    And here is the flag that we wanted to capture to gain access into bandit level 12.
  4. Let's break this command down. cat data.txt returns the string inside of the data.txt file that we want to manipulate. | tr 'a-z' 'n-za-m' the pipe takes the cat returned value and pumps it into tr, then you define what to look for and lowercase letters only looks for lowercase letters. Add a space, the space is very important, then what to replace the character with. Then you do the same with uppercase letters. I learned this is called ROT13 and ROT13 is special is the since that it is the halfway point in the english alphabet so it doesn't matter is 'a-z' is first or 'n-za-m' is first.
  5. And don't forget to use the login username of bandit12 for the next level.

* Please note that the password is not shown as per Over The Wire's request.

Learned:

  • How to rotate a string 13 characters and how to decode a ROT13 string with the tr command.
